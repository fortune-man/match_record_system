구현할 기능 목록 정리
1. 불러 오기 
    1. 기존 선수 경기 기록을 조회 합니다. 
    2. 없으면 사용자가 직접 입력 선택도 가능합니다
2. 경기 진행 설정
    1. 총 12열이고 ‘인상’, ‘용상’ 열은 각각 1차, 2차, 3차 경기 진행 순서로 추가 분리 필요합니다.
    2. 성별 | 구분  접수번호 | 성명 | 시도 | 소속 | 생년월일 | 학년 | 체중 | 인상 ( 1차 | 2차 | 3차 | 결과 | 등위 ) | 용상 ( 1차 | 2차 | 3차 | 결과 | 등위 ) | 합계 | 등위 | 비고 | 인쇄 옵션
    3. 성별 - [] 남, [] 여 커서 선택
    4. 구분 - [] 일반, [] 고, [] 중 커서 선택
    5. 체급 - 전체, 선택 ([] 49kg, [] 59kg, [] 61kg, [] 67kg, [] 73kg, [] 81kg, [] 89kg, [] 96kg, [] 102kg, [] 102+ )
    6. 선택 완료 및 경기 시작 버튼 활성화 / 선택 수정으로 커서 클릭시 상태 변경 가능
3. 경기 진행 : 용상
    1. 접수 번호 순서로 각 선수행 경기 진행
    2. 1차, 2차, 3차 진행 기록 수치 입력
    3. 입력값 적용
    4. 성공 / 실패 버튼 활성화 -> 푸른색 / 적색 빗금으로 상태 변경
    5. 경기 순번 행에 상태 변경 결과 전달 -> 저장
    6. 1차, 2차, 3차 경기 결과 중 성공 결과가 가장 큰 값을 결과 열 정보로 전달 -> 실시간 상태 변경
    7. 결과 열 정보를 인상 선수 기록 기준 내림차순으로 계산된 결과를 등위 열에 전달 -> 실시간 상태 변경
    8. 결과 열 정보를 합계 열에 전달 -> 저장
    9. 접수 번호 끝까지 동일 작업 반복
4. 경기 진행 : 인상
    1. 용상과 동일 작업 반복
    2. 용상 결과 추가 시 합계 열에 기록 결과 전달 -> 합산하여 상태변경
    3. 저장
5. 경기 종료 : 탭이동
    1. 해당 경기 진행된 체급 수 만큼 인쇄 옵션 생성
    2. 생성된 옵션 별 경기 기록 조회
    3. Excel 내보내기
    4. 옵션별 / 전체 내용 선택 가능
